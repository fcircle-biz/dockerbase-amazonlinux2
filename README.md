# dockerbase-amazonlinux2

## 実行
  ```
  > docker-compose up -d
  > docker-compose exec amzn-linux-2 /bin/bash
  bash-4.2# su - ec2-user
  [ec2-user@XXXXX ~]$ pwd
  /home/ec2-user
  [ec2-user@XXXXX ~]$
  [ec2-user@XXXXX ~]$ exit
  logout
  bash-4.2# exit
  exit  
  > 
  ```
